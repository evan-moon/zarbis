> [Live Demo](https://zarbis.herokuapp.com/)

## Installation
```bash
git clone https://github.com/evan-moon/zarbis.git
cd zarbis
yarn
```

```bash
yarn dev # 개발용 서버

yarn build # 빌드

yarn start # Express on
```
